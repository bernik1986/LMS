# Контроль качества импорта WordPress / Tutor LMS

## Быстрая проверка

Открыть в админке:

- `Админ -> Файлы`
- `Админ -> Отчеты`
- `Админ -> Тесты`

## Что проверять после импорта

1. Битые локальные файлы

   Раздел `Админ -> Файлы -> Материалы, где файл не найден`.

2. Непривязанные видео

   Раздел `Админ -> Файлы -> Видео в uploads без привязки к урокам`.

3. Пустые курсы и уроки

   Раздел `Админ -> Файлы -> Пустые импортированные курсы и уроки`.

4. Количество уроков, материалов и видео

   Раздел `Админ -> Файлы -> Импортированные курсы WordPress/Tutor LMS`.

5. Тесты

   Раздел `Админ -> Тесты`.

## Экспорт отчета

В админке: `Админ -> Файлы -> Экспорт отчета импорта`.

Прямая ссылка:

```text
/admin/files/import-report.csv
```

CSV содержит:

- курс;
- WordPress ID;
- количество уроков;
- количество материалов;
- количество видео;
- количество битых файлов;
- количество пустых уроков.

## Ручная привязка видео

1. Открыть `Админ -> Файлы`.
2. Найти видео в блоке непривязанных файлов.
3. Выбрать урок в dropdown.
4. Указать название материала.
5. Нажать `Привязать`.

Система создаст новый материал типа `video` в выбранном уроке.

## Массовая автопривязка

Кнопка `Автопривязать видео по названию` пытается сопоставить имя видео с названием курса, урока и материалов.

Правила:

- обрабатываются только непривязанные видео;
- создается новый обязательный материал типа `video`;
- в source материала записывается `auto_video_match`;
- если похожего урока не найдено, файл не трогается.

Перед массовой привязкой рекомендуется:

```powershell
npm.cmd run backup:data
```

## Когда нужна ручная проверка

Ручная проверка нужна, если:

- курс без уроков;
- урок без материалов;
- видео называется слишком общо, например `part-1.mp4`;
- курс импортирован без теста;
- студент завершил старый курс в Tutor LMS, но в новой LMS нет сертификата.
