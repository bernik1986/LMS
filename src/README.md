# Next.js UI-каркас

Эта папка **не является рабочим приложением**.

| Что искать | Где |
|------------|-----|
| Рабочая платформа (заявки, вход, админка, тесты, сертификаты) | `scripts/lms-server.mjs` |
| Запуск | `npm run dev` → http://127.0.0.1:3000 |
| Данные | `data/db.json` или PostgreSQL через Prisma |
| Этот каркас | `npm run scaffold:dev` → http://127.0.0.1:3000 (конфликт порта!) |

`src/app/` — прототип интерфейса на mock-данных из `src/lib/mock-data.ts` для возможной будущей миграции на Next.js. Логику и API меняйте в `scripts/lms-server.mjs`, а не здесь.
